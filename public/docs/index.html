Introduction

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

Authenticate request to this API's enpoints by sending an Authotization Header with the value "Bearer {TOKEN}"

Session Token

To authenticate users, you can use the following endpoint:

GET http://{baseUrl}/sanctum/token

Parameters

  • email: The user's email address.
  • password: The user's password.

First User

  • email: yoda@gmail.com
  • password: 12345678

Books

Get books.

requires authentication

this endpoint provides the books of a user

Example request:
curl --request GET \
    --get "http://localhost/api/books?user_id=1" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost/api/books"
);

const params = {
    "user_id": "1",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());

Create Book.

requires authentication

It returns the created book.

Example request:
curl --request POST \
    "http://localhost/api/books" \
    --header "Content-Type: multipart/form-data" \
    --header "Accept: application/json" \
    --form "title=velit"\
    --form "description=Sit et repellendus est vero debitis vel sed qui."\
    --form "credit=13"\
    --form "user_id=9"\
    --form "image=@/private/var/folders/_9/7jj6bsk14kjb_x4rfwxlwdn40000gn/T/php1Doy9H" 
const url = new URL(
    "http://localhost/api/books"
);

const headers = {
    "Content-Type": "multipart/form-data",
    "Accept": "application/json",
};

const body = new FormData();
body.append('title', 'velit');
body.append('description', 'Sit et repellendus est vero debitis vel sed qui.');
body.append('credit', '13');
body.append('user_id', '9');
body.append('image', document.querySelector('input[name="image"]').files[0]);

fetch(url, {
    method: "POST",
    headers,
    body,
}).then(response => response.json());
