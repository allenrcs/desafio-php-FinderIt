# Desafío PHP Laravel

Proyecto Laravel 9.x

## Requisitos

  - PHP (8.1.14)
  - Mysql
  - Composer
  - NPM

## Instalación

Clonar este repositorio, instalar dependecias de composer

```bash
composer install
```

instalar dependencias de NPM

```bash
npm install
```

Base de datos

Creación tablas

```bash
php artisan migrate
```

Carga inicial de datos

```bash
php artisan db:seed
```

Compilación

```bash
npm run dev
```

Iniciar servidor de desarrollo

```bash
php artisan serve
```
